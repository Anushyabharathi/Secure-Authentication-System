Secure Authentication System:

PROJECT OVERVIEW:
This is a simple and secure authentication system built using Python, Flask, and SQLite. 
It allows users to register, login, and access a dashboard. Passwords are hashed with bcrypt for security. 
This project demonstrates secure user authentication and session management in a lightweight Flask app.

----------------------------------------------------------------------------------------------------------------------------------------

TECHNOLOGIES USED:
      * Python
      * Flask
      * SQLite
      * bcrypt

----------------------------------------------------------------------------------------------------------------------------------------

FEATURES:
* User registration with hashed passwords
* User login with session management
* Dashboard page visible only to logged-in users
* Logout functionality
* Styled pages with error/success messages
  
-----------------------------------------------------------------------------------------------------------------------------------------

WORKFLOW:
    1. Register: Users create an account. Username must be unique.
    2. Login: Users log in using username and password.
    3. Dashboard: Logged-in users see a welcome page.
    4. Logout: Users can log out, ending their session.

------------------------------------------------------------------------------------------------------------------------------------------

SECURITY FEATURES:
    * Passwords are hashed with bcrypt
    * Session-based login ensures dashboard is protected
    * Prevents duplicate usernames
